# Themes

Craft ships with 25 built-in color themes covering the popular terminal palettes. The default theme is **dracula**.

## Switching

Run `/theme` to open the theme picker. Navigate to preview a theme live, then press Enter to apply and save it. Your choice persists across sessions.

## Bundled Themes

`ayu_dark`, `carbonfox`, `catppuccin_frappe`, `catppuccin_latte`, `catppuccin_macchiato`, `catppuccin_mocha`, `dracula`, `everforest_dark`, `fleet_dark`, `github_dark`, `gruvbox`, `gruvbox_light`, `kanagawa`, `material_darker`, `monokai_pro`, `night_owl`, `nightfox`, `nord`, `onedark`, `rose_pine`, `solarized_dark`, `solarized_light`, `tokyonight`, `vscode_dark_plus`, `zenburn`.

Themes use a Helix-style TOML format (`[palette]`, `[ui]`, and syntax-scope tables) and are compiled into the binary, so custom theme files are not currently supported.
